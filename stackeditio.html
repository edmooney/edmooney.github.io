Excellent markdown editor stackedit.io

Date Wed 29 January 2014 Tags webdev / tools markdown

stackedit.io

I often look for browser based tools that are cross-platform, somewhat lightweight and easy to use. I happen to come across stackedit.io that's a nice split screen markdown editor with export options and clean user interface. It's worth trying it out.
